# Introduction
This project is to create a 3D graphical view inside a maze from a first player perspective using raycasting principles. The graphic is rendered using miniLibx library.

# Setup
Git clone the project
```
git clone git@github.com:LumiLu99/cub3d.git
```
Compile the project
```
make
```
Run the program with the valid .cub file in map directory
```
./cub3D map/default.cub
```

# Control
`W` `A` `S` `D` - control player movement in the maze \
Arrow `Left <-` `Right ->` - control player look left and right \
Button `Esc` or `X` on window frame - quit the program

# Bonus Feature
- Minimap
- Wall collisions
- Animated sprite
- Mouse movement to rotate the point of view
